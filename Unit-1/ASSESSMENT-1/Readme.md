# Artificial Intelligence – Assessment 1

## Title
Analytical Problem Solving using Artificial Intelligence

## Objective
To understand and implement fundamental Artificial Intelligence concepts and search algorithms through practical problem-solving using Python.

## Software Used
Python 3.13

## Tools Used
Python IDLE

## Algorithms / AI Concepts Used
- Breadth First Search (BFS)
- Uniform Cost Search (UCS)
- Backtracking Search
- Intelligent Agent

## Problems Implemented
1. Water Jug Problem using Breadth First Search (BFS)
2. Mars Rover Agent Simulation
3. 8-Queens Problem using Backtracking Search
4. Ola Cab Booking Agent
5. Uniform Cost Search (UCS) for Least-Cost Path Finding

## Result
Successfully implemented all five Artificial Intelligence programs using Python. The programs demonstrated the application of search algorithms, intelligent agents, and constraint satisfaction techniques to solve various AI problems, including path finding, scheduling, optimization, and autonomous decision-making.
