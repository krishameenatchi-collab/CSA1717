# Artificial Intelligence – Assessment 2

## Title
Analytical Problem Solving using Artificial Intelligence Search Algorithms

## Objective
To formulate and solve Artificial Intelligence search problems using appropriate search algorithms and intelligent agent concepts.

## Software Used
Python 3.13

## Tools Used
Python IDLE

## Algorithms Used
- Backtracking Search
- Breadth First Search (BFS)
- Uniform Cost Search (UCS)

## Problems Implemented
1. Doctor Shift Scheduling using Backtracking Search
2. Robot Grid Navigation using Breadth First Search (BFS)
3. Autonomous Rescue Robot using Uniform Cost Search (UCS)

## Result
Successfully implemented and tested all three Artificial Intelligence problems using Python. The algorithms generated valid solutions by satisfying the given constraints, finding the shortest path, and determining the least-cost path for the rescue robot.
